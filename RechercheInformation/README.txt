Version 1 du projet de recherche d'information:

##Résumé##
Calucule du term frequency basé sur le simple nombre d'occurences des termes rechercher dans les documents.

##Lancer le projet##
- Ouvrir le projet dans eclipse
- Lancer le Main
- Regarder les résultats affciher dans la console

Version 2 du projet de recherche d'information:

##Résumé##
Calucule du term frequency basé sur le nombre d'occurences des termes rechercher dans les documents par rapport au nombre de terme que possède celui-ci.

##Lancer le projet##
- Ouvrir le projet dans eclipse
- Lancer le Main
- Regarder les résultats affciher dans la console
