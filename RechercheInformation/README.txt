Projet de recherche d'information:

##Lancer le projet##

- Ouvrir le projet dans eclipse
- Lancer le Main (main.java) de la version à tester (voir le rapport pour plus de détails)
main(1,0) : version 1 : tf simple
main(2,0) : version 2 : tf pondéré
main(3,0) : version 3 : tf idf
main(4,0) : version 4 : cosinus
main(5,0) : version 5 : prise en compte des balises

Pour tester la partie web sémantique, lancer :
main(3,1) : version 3 : tf idf, reformulation simple
main(4,1) : version 4 : cosinus, reformulation simple
main(5,1) : version 5 : prise en compte des balises, reformulation simple

main(3,2) : version 3 : tf idf, reformulation avec poids
main(4,2) : version 4 : cosinus, reformulation avec poids
main(5,2) : version 5 : prise en compte des balises, reformulation avec poids

- Regarder les résultats affichés dans la console
